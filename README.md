# Sports-Fitness
A MERN STACK application for booking venues.
## Usage

### Clone
Clone the Project

```sh
$ git clone https://github.com/tanishha/Sports-Fitness.git
$ cd Sports-Fitness
```

### Installation

Install the dependencies

```sh
$ npm install
```

### Run

This will run your dev server at http://localhost:3000

```sh
$ npm start
```
